EMOJIS IN HTML

Emojis are characters from the UTF-8-character set: 😄 😍 💗

What are Emojis?

  • Emojis look like images, or icons, but they are not.
  • They are letters (characters) from the UTF-8 (Unicode) character set.
  • The HTML charset Attribute
  • To display an HTML page correctly, a web browser must know the character set used in the page.
  • This is specified in the tag:
 <meta charset="UTF-8">

Emoji Characters

  • Emojis are also characters from the UTF-8 alphabet:
  • 😄 is 128516
  • 😍 is 128525
  • 💗 is 128151

Example:

 <!DOCTYPE html> 
                 <html lang="en">
                <head>
                 <meta charset="UTF-8">
                 <meta name="viewport" content="width=device-width, initial-scale=1.0">
                 <title> Mathematical Symbols</title>
                 </head>
                 <body>
                 <p>&plusmn; - ± (plus-minus)</p> 
                 <p>&times; - × (multiplication)</p> 
                <p>&divide; - ÷ (division)</p>
                 <p>&equals; - = (equals)</p> 
                 <p>&ne; - ≠ (not equal)</p>
                 <p>&le; - ≤ (less than or equal)</p> 
                <p>&ge; - ≥ (greater than or equal)</p>
                 <p>&sum; - ∑ (summation)</p>
                 <p>&prod; - ∏ (product)</p>
                 <p>&radic; - √ (square root)</p>
                 <p>&infin; - ∞ (infinity)</p>
                 <p>&int; - ∫ (integral)</p>
                 <p>&sim; - ∼ (similar to)</p> 
                <p>&pi; - π (pi)</p>
                 <p>&theta; - θ (theta)</p> 
                 <p>&lambda; - λ (lambda)</p>
                 </body>
                 </html>
