HTML ENTITIES

• HTML entities, also known as character entities or HTML character references, are codes used in HTML to represent special characters, symbols, and reserved characters that have special meanings or functions in HTML.

1. ( ) - non-breaking space:- Represents a non-breaking space, which prevents line breaks or word wraps at that point.

2. (<) - less than:-Represents the less-than sign `<`.

3. (>) - greater than:- Represents the greater-than sign `>`.

4. (&) - ampersand:- Represents the ampersand `&`.

5. ( ")- double quotation mark:- Represents the double quotation mark `"`.

6. (') - single quotation mark (apostrophe):- Represents the single quotation mark or apostrophe `'`.

7. (¢) - cent:- Represents the cent symbol ¢.

8. (£) - pound:- Represents the pound sterling symbol £.

9. (¥) - yen:- Represents the yen symbol ¥.

10. (€) - euro:- Represents the euro symbol €.

11. (©) - copyright:- Represents the copyright symbol ©.

12. (®) - registered trademark:- Represents the registered trademark symbol ®
