HTML SYMBOLS

Mathematical Symbols:

1. Basic Operators:
• `+` Addition: &plus; or &#43;
Example: 5 &plus; 3 displays as "5 + 3."
• `-` Subtraction: &minus; or &#8722;
Example: 10 &minus; 4 displays as "10 - 4."
• × Multiplication: &times; or &#215;
Example: 6 &times; 9 displays as "6 × 9."
• ÷ Division: &divide; or &#247;
Example: 12 &divide; 4 displays as "12 ÷ 4."
• = Equals: &equals; or `&#61;
Example: 2 &plus; 2 &equals; 4 displays as "2 + 2 = 4."

2. Exponents and Superscripts:
• ² Squared: &sup2; or &#178;
Example: x² displays as "x²."
• ³ Cubed: &sup3; or &#179;
Example: y³ displays as "y³."
• nⁿ Superscript n: &nSup; (Replace "Sup" with the desired number)
Example: a⁴ displays as "a⁴."
• xⁿ Custom Superscript: `&xsupn;` (Replace "x" with the base and "n" with the exponent)
Example: b⁵ displays as "b⁵."

3. Square Root and Radicals:

• √ Square Root: &radic; or &#8730;
Example: √16 displays as "√16."
• ⁿ√ Custom Root: `&nsqrt;` (Replace "n" with the desired root)
Example: ⁵√243 displays as "⁵√243."

4. Fractions:
• ½ One Half: &frac12; or &#189;
Example: `¾ + ½` displays as "¾ + ½."
• ¼ One Fourth: &frac14; or &#188;
Example: 1/4 + 3/4 displays as "1/4 + 3/4."
• ¾ Three Fourths: &frac34; or `&#190;
Example: ⅔ + ¾ displays as "⅔ + ¾."

5. Greek Letters:
• α Alpha: &alpha; or &#945;
Example: Δ = &alpha; + &beta; displays as "Δ = α + β."
• β Beta: &beta; or &#946;
Example: θ = &beta; × &gamma; displays as "θ = β × γ."
• Σ Sigma (Summation): &Sigma; or &#931;
Example: &Sigma; i = 1 to n represents summation. • π Pi: &pi; or &#960;
Example: π ≈ 3.14159 displays an approximation of pi.
• θ Theta: &theta; or &#952;
Example: θ = 45° displays an angle in degrees.

6. Inequality Symbols:
• ≤ Less Than or Equal To: &leq; or `&#8804;`
Example: x &leq; y displays as "x ≤ y."
• ≥ Greater Than or Equal To: &geq; or &#8805;
Example: a &geq; b displays as "a ≥ b."
• ≠ Not Equal To: &ne; or &#8800;
Example: x &ne; y displays as "x ≠ y."

7. Logical Operators:
• ∧ Logical AND: &and; or &#8743;
Example: A &and; B displays as "A ∧ B."
• ∨ Logical OR: &or; or `&#8744;
Example: X &or; Y displays as "X ∨ Y."
• ¬ Logical NOT: &not; or &#172;
Example: ¬P displays as "¬P."

8. Infinity:
• ∞ Infinity: &infin; or &#8734;
Example: 1/∞ = 0` represents the concept of infinity.

9. Summation and Integration:
• ∑ Summation: &sum; or &#8721;
Example: ∑(i = 1 to n) xᵢ represents summation.
• ∫ Integral: &int; or &#8747;
Example: ∫(0 to 1) f(x) dx represents integration.

10. Set Theory:
• ∈ Element Of: &isin; or &#8712;
Example: x ∈ A displays as "x ∈ A."
• ∉ Not an Element Of: &notin; or &#8713;
Example: y ∉ B displays as "y ∉ B."
• ∪ Union: &cup; or &#8746;
Example: A ∪ B represents the union of sets A and B.
• ∩ Intersection: &cap; or &#8745;
Example: X ∩ Y represents the intersection of sets X and Y.
